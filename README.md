# 🎟️ Платформа для организации мероприятий (+Vibe)

Веб-приложение для автоматизации продажи билетов, управления мероприятиями и бронирования мест с интерактивной схемой зала и генерацией QR-билетов.

## 🎯 Назначение проекта

Проект предназначен для организаторов мероприятий (концертов, шоу) и их посетителей. Он заменяет очереди в кассы и телефонное бронирование на современный онлайн-инструмент с прозрачной системой оплаты и контроля посещаемости.

**Ключевые возможности:**
- Регистрация и JWT-авторизация
- Просмотр афиши с фильтрацией и поиском
- Интерактивная схема зала (выбор конкретных мест)
- Генерация QR-кодов для купленных билетов
- Личный кабинет с историей покупок
- Администрирование мероприятий и мест

## 🛠️ Стек технологий

| Компонент       | Технологии                                                                 |
|----------------|----------------------------------------------------------------------------|
| Backend        | C#, ASP.NET Core MVC, REST API                                             |
| Frontend       | React.js, React Router, Framer Motion, Tailwind CSS                        |
| База данных    | Microsoft SQL Server (LocalDB / SSMS)                                      |
| Аутентификация | JWT-токены, BCrypt (хеширование паролей)                                   |
| Почта          | SMTP-сервис (приветственные письма, верификация)                           |
| Инструменты    | Visual Studio Code, Git, SQL Server Management Studio                      |
